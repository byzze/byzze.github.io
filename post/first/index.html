第一个博客网站

下载hugo

  • 确保已经安装了go语言环境
  • 在终端执行命令go install github.com/gohugoio/hugo@latest
  • 安装成功

创建网站，引入主题

mkdir ~/hugo
cd ~/hugo
hugo new site blog
cd blog 
git submodule add <https://github.com/WingLim/hugo-tania> themes/hugo-tania

步骤 git submodule 执行失败两点问题**

  1. https请求修改为项目的git@请求
  2. 在个人github创建对应仓库 blog ，初始化，push相关操作 参考地址：https://themes.gohugo.io/themes/hugo-tania/

启动服务 

1

hugo server

访问地址 http://localhost:1313/
